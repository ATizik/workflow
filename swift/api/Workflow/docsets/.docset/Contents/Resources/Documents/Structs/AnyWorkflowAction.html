AnyWorkflowAction

public struct AnyWorkflowAction<WorkflowType> : WorkflowAction where WorkflowType : Workflow

A type-erased workflow action.

The AnyWorkflowAction type forwards apply to an underlying workflow action, hiding its specific underlying type, or to a closure that implements the apply logic.
