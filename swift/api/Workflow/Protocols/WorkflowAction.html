WorkflowAction

public protocol WorkflowAction

Conforming types represent an action that advances a workflow. When applied, an action emits the next state and / or output for the workflow.

  • WorkflowType

    The type of workflow that this action can be applied to.

    Declaration

    Swift

    associatedtype WorkflowType : Workflow
  • apply(toState:)

    Applies this action to a given state of the workflow, optionally returning an output event.

    Declaration

    Swift

    func apply(toState state: inout WorkflowType.State) -> WorkflowType.Output?

    Parameters

    state

    The current state of the workflow. The state is passed as an inout param, allowing actions to modify state during application.

    Return Value

    An optional output event for the workflow. If an output event is returned, it will be passed up the workflow hierarchy to this workflow’s parent.
